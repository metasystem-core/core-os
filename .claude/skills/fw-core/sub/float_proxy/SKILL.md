# SKILL — Float Strategy Detector

## Função

Detectar oportunidades defensivas de gestão de float sem virar alavancagem.

## Contexto

Estratégia de usar cartão/reembolsos/conta remunerada para rendimento marginal, mantendo liquidez e pagamento integral.

## Regras

- pagar fatura integralmente;
- nunca usar rotativo;
- não depender de rendimento futuro para pagar;
- só usar reembolso líquido antes do vencimento;
- manter liquidez total;
- interromper se houver risco de atraso;
- não usar limite perto do máximo de forma recorrente;
- se gerar estresse, risco ou dependência, invalida.

## Quando usar

- reembolso;
- cartão de crédito;
- Mercado Pago;
- CDI;
- conta remunerada;
- antecipação;
- pagamento futuro;
- liquidez.

## Saída

- float válido;
- float inválido;
- risco detectado;
- alternativa segura;
- ganho marginal estimado, se houver dados.

## Regra

Float defensivo não é alavancagem.
