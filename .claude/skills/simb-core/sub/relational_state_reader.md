# Relational State Reader

## Função

Identificar sinais de estado alterado em terceiros
numa interação relacional — antes de agir, responder ou concluir.

## Quando usar

- antes de uma conversa difícil
- após uma interação confusa
- quando a resposta do outro parece desproporcional
- quando há tensão relacional não nomeada

## Sinais observáveis

| Sinal | O que pode indicar |
|---|---|
| Silêncio prolongado | Sobrecarga, medo, processamento |
| Resposta curta e seca | Raiva contida, fechamento |
| Apressamento | Ansiedade, fuga |
| Excesso de explicação | Culpa, defesa antecipada |
| Mudança de assunto | Evitação, desconforto |
| Irritabilidade desproporcional | Sobrecarga de outro contexto |
| Generosidade excessiva | Culpa, compensação |

## Regra

Leitura relacional é operacional — serve para decidir
como agir, não para diagnosticar o outro.

O leitor não é terapeuta. É operador que precisa
de informação para agir com mais precisão.

## Onde pode falhar

- Projetar estado próprio no outro
- Leitura confirmatória — ver o que quer ver
- Usar leitura para manipular em vez de conectar
- Concluir sem verificar diretamente com a pessoa
