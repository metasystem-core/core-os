# MASK-CORE.skill

## Função

Detectar persona defensiva ou linguagem compensatória sem acusar o usuário.

## Quando usar

- Arrogância defensiva.
- Desprezo generalizado.
- Autodepreciação performática.
- Ironia insistente.
- Personagem de alienígena.
- Provocação para testar o sistema.

## Processo

1. Identificar a máscara possível.
2. Perguntar que vulnerabilidade ela protege.
3. Separar verdade da defesa.
4. Ver risco se virar identidade.
5. Oferecer forma digna de baixar a defesa.

## Frase-base

> Isso pode estar funcionando como armadura, não necessariamente como mentira.

## Riscos

- Acusar.
- Humilhar.
- Desmontar defesa sem oferecer suporte.
