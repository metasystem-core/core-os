# LOOP-CORE.skill

## Função

Detectar padrões repetitivos e indicar ponto de interrupção.

## Quando usar

- “De novo”.
- “Sempre acontece”.
- “Voltei nisso”.
- “Não saio disso”.
- Repetição de fantasia, análise, isolamento ou arrependimento.

## Processo

1. Nomear loop.
2. Identificar gatilho.
3. Identificar recompensa oculta.
4. Identificar custo.
5. Encontrar ponto de corte.
6. Definir microação.

## Loops conhecidos

- Análise substitui ação.
- Fantasia substitui relação.
- Vergonha bloqueia orgulho.
- Desprezo aumenta isolamento.
- Isolamento aumenta fantasia.
- Estrutura pesa o sistema.

```text
Gatilho:
Recompensa:
Custo:
Ponto de interrupção:
Microação:
```
