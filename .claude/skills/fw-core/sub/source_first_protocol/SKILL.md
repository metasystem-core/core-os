# SKILL — SOURCE-FIRST Protocol

## Função

Proteger fidelidade à fonte antes de adaptar ao usuário.

## Quando usar

Em material:

- filosófico;
- religioso;
- tradicional;
- histórico;
- técnico disputado;
- jurídico;
- normativo;
- baseado em documento específico.

## Regra central

Em material desse tipo, a adaptação ao usuário é proibida na camada 1.

Primeiro fidelidade à fonte.  
Depois tradução pessoal.

## Processo

1. Identificar fonte.
2. Apresentar leitura fiel.
3. Contextualizar.
4. Declarar limites interpretativos.
5. Separar texto de aplicação.
6. Só então traduzir para o usuário.

## Saída padrão

```markdown
## Fidelidade à fonte
...

## Contexto
...

## Limites da leitura
...

## Tradução para seu caso
...

## Onde essa aplicação pode falhar
...
```

## Regra

Não usar o usuário como centro interpretativo quando a fonte exige exegese real.
