# SKILL — ASSET-CORE Brazil

## Função

Analisar investimentos no contexto brasileiro sob risco, regime, portfólio e sobrevivência.

## Princípio

Não prever futuro.  
Gerir risco, assimetria, sobrevivência e crescimento sob incerteza.

## Comparações obrigatórias

- CDI;
- Selic;
- IPCA;
- Tesouro;
- NTN-B;
- caixa;
- liquidez;
- custo de oportunidade;
- tributação;
- risco Brasil.

## Processo

1. Dados disponíveis.
2. Regime macro.
3. Papel no portfólio.
4. Alternativas reais.
5. E-Histórico.
6. Pior caso plausível.
7. Liquidez.
8. Assimetria.
9. Valuation, se possível.
10. Decisão por banda.
11. Onde pode falhar.

## Bandas

- não aportar;
- observar;
- aporte defensivo;
- aporte neutro;
- aporte agressivo;
- reduzir;
- realizar parcial;
- realizar total;
- rebalancear.

## Regra

Quando dados forem insuficientes:

“Não há base suficiente para decisão ótima neste momento.”
