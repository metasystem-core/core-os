# CON-014 — Maré Solar (Solar Tide)

**Fonte:** Rudhyar, *A Prática da Astrologia*, Passo 6  
**Tipo:** Conceito estrutural — modelo de significado planetário  
**Data:** 2026-06-12  

---

## Definição

O sistema solar como organismo integrado onde os planetas não são forças autônomas, mas "transformadores" (prismas) da energia solar que se difunde em duas direções: para fora (involução/descida) e para dentro (evolução/ascensão).

**MARÉ PARA FORA (involução):**  
Sol → Mercúrio (eletricidade celular/transmissão) → Vênus (campo magnético/forma-semente arquetípica) → Terra/Lua → Marte (ação orgânica/ego-impulso) → Júpiter (assimilação/expansão social) → Saturno (estrutura/limitação/consciência-do-lugar)

**MARÉ PARA DENTRO (evolução):**  
Saturno → Júpiter (compaixão) → Marte (fé/ação regenerada) → Vênus (valor/amor) → Mercúrio (pensamento-semente) → Sol (retorno ao centro)

## Implicação operacional

O significado de cada planeta não é um atributo isolado — é sua posição na maré solar que define seu papel. Mercúrio não é "apenas comunicação": é eletricidade celular do sistema solar, transmissão nervosa da energia cósmica para o organismo terrestre. Vênus não é "apenas amor": é o campo magnético que cria a forma-semente onde o impulso solar se cristaliza.

## Relação com IRIS

**GAP-R12**: IRIS opera com catálogo de atributos planetários isolados. A maré solar é modelo estruturalmente superior: cada planeta tem significado relacional, não absoluto.
