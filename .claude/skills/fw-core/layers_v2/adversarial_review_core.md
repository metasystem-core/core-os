# SKILL — Adversarial Review Core

## Função

Atacar uma tese, decisão, modelo ou plano para revelar fragilidades.

## Quando usar

- tese forte demais;
- decisão importante;
- plano profissional;
- valuation;
- jurídico;
- investimento;
- produto;
- translação de domínio;
- convicção excessiva.

## Perguntas

1. Como essa tese falha?
2. O que estou ignorando?
3. Que dado desmente isso?
4. Qual premissa é fraca?
5. Qual incentivo distorce minha leitura?
6. Como eu atacaria essa conclusão?
7. O que parece coerente só porque eu quero que seja?
8. Qual é o pior argumento contra isso?
9. Qual é o erro raro, mas fatal?
10. Em que regime isso quebra?
