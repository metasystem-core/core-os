# SUBSKILL — Anchorage Plan Builder

## Função

Construir ou revisar plano de ancoragem para acesso por cordas.

## Itens obrigatórios

- identificação dos pontos;
- tipo de estrutura;
- critério de validação;
- redundância;
- equalização;
- ângulos;
- proteção de aresta;
- linha de trabalho;
- linha de segurança;
- rota das cordas;
- desvios/fracionamentos;
- interferências;
- inspeção pré-uso;
- croqui textual;
- fotos, quando disponíveis;
- aprovação/liberação.

## Bloqueios

- ponto sem critério de validação;
- ponto não identificado;
- estrutura duvidosa;
- ausência de redundância quando aplicável;
- ângulo crítico sem avaliação;
- aresta sem proteção;
- interferência com processo/equipamento;
- ausência de rota segura de cordas.

## Regra

Ponto de ancoragem sem validação/documentação/critério definido não deve ser tratado como ponto liberado.
