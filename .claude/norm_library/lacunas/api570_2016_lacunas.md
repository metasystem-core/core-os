# API 570 - Lacunas e alertas de ingestão

## Lacuna crítica de vigência
- O arquivo carregado é API 570, 4th Edition, February 2016.
- Pesquisa oficial indicou existência da 5th Edition, February 2024.
- Uso em campo como referência conclusiva deve confirmar qual edição é exigida pelo contrato, cliente, procedimento interno ou jurisdição.

## Lacunas de fonte
- Não foram incorporados Addendum 1 (May 2017), Addendum 2 (March 2018) e Errata 1 (April 2018), caso sejam exigidos para o uso da 4th Edition.
- Não foi incorporada a 5th Edition 2024.
- O PDF bruto não foi empacotado neste ZIP por política de copyright/licença.

## Lacunas técnicas para uso operacional
- Para cálculo real de vida remanescente: faltam t_initial, t_previous, t_actual, t_required, datas e confirmação de ponto repetível.
- Para classificação de serviço: faltam fluido, temperatura, pressão, localização, toxicidade/inflamabilidade e consequência de vazamento.
- Para uso em reparos: faltam WPS/PQR, qualificação de soldador, aprovação do inspetor/engenheiro e procedimento do owner/user.
- Para FFS: faltam dimensões do dano, material, cargas, pressão/temperatura, taxa futura e código aplicável.

## Política de lacuna
Se a consulta depender desses dados, a resposta deve declarar: "Não há base suficiente para conclusão operacional; listar dados faltantes e próximos passos".
