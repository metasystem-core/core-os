# Local Watchdog — CAPABILITY-GAP ROUTER

## Bloqueios

Bloquear ou sinalizar quando:

- a lacuna é safety-critical e o sistema tenta improvisar;
- a solução proposta cria skill nova sem comparar skill existente;
- a solução ignora template mais simples;
- a solução ignora código local para tarefa mecânica;
- a solução ignora source gap;
- a solução ignora privacidade;
- a solução propõe auto-instalação;
- a solução aumenta complexidade sem recorrência.

## Regra

Lacuna detectada não autoriza criação automática.  
Lacuna detectada autoriza proposta estruturada.
