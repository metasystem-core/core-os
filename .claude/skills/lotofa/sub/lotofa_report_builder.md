# Sub-skill: lotofa_report_builder

## Função específica
Gerar relatórios claros e exportáveis sobre sets, versões, auditorias e comparações.

## Quando consultar
Quando o usuário pedir tabela, matriz, comparação, auditoria ou arquivo para exportação.

## Estrutura recomendada

1. Identificação do modelo/versão.
2. Dataset usado.
3. Parâmetros relevantes.
4. Jogos gerados ou avaliados.
5. Métricas principais.
6. Comparação com baseline.
7. Interpretação crítica.
8. Onde pode falhar.
9. Arquivos gerados.

## Formatos

- Markdown;
- CSV;
- JSON metadata;
- ZIP para migração.

## Regra crítica

Relatório deve ser legível, rastreável e não promocional. Não vender certeza.
