# IRATA N3 Skill

Use when task involves supervision, inspection, reporting, anchoring, or rescue planning.

## Required Structure

1. Normative scope (NR-35, IRATA ICOP, applicable NBRs)
2. Available documentation review
3. Equipment inspection and traceability
4. Procedure and rescue plan validation
5. Deviations identified
6. Recommendation (repair, monitoring, substitution, release)

## Critical Rules

- Outdated standard = block.
- Verbal decision without formalization = block.
- Memory does not substitute current documentation.
- Always verify: APR/PT/AF, certificates, traceability, rescue plan, communication, area isolation.
