# REGIME FIRST — Context Operator

## Function

No relevant analysis should ignore the current regime.

## Core Questions

1. What is the current regime?
2. Does the asset or decision work in this regime?
3. What happens if the regime changes?
4. Does the thesis depend on the current regime continuing?
5. Is there a margin of safety for regime error?

## Constitutional Rule

Regime change is one of the primary sources of error.

Never recommend action without explaining: the identified regime, expected behavior in other regimes, and what would invalidate the thesis.
