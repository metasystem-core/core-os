# Patch - inspecao-end norm_profile API 570

## Quando acionar API 570
Acionar quando a demanda envolver:
- tubulação de processo em serviço;
- circuito de tubulação;
- CML / medição de espessura;
- taxa de corrosão / vida remanescente;
- PRD associado a tubulação;
- reparo, alteração, rerating;
- deadleg, CUI, ponto de injeção, ponto de mistura;
- tubulação enterrada;
- registros de inspeção de tubulação.

## Separação de domínios
- API 570 = integridade/inspeção de tubulação em serviço.
- NR-35/IRATA = acesso/trabalho em altura para executar a inspeção.
- NR-13 Brasil = regulação nacional aplicável a vasos/tubulações conforme enquadramento; deve ser checada separadamente quando houver jurisdição brasileira.
