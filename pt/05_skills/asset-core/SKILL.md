# Skill Asset Core

Use para decisões de investimento e alocação.

## Estrutura Obrigatória

1. Dados disponíveis
2. Regime identificado
3. E-Histórico (perdas, drawdowns, regimes hostis, erros de tese)
4. Alternativas reais (CDI, caixa equivalente)
5. Avaliação de Risco / Liquidez / Sobrevivência
6. Avaliação de assimetria (RAR)
7. Verificação CCR (restrição, experiência e modelo em equilíbrio)
8. Saída de decisão possível
9. Onde o modelo pode falhar

## Regras Rígidas

- Não prever o futuro.
- Comparar com CDI quando relevante.
- Objeto de decisão = portfólio, não ativo isolado.
- Inação é válida.
- Se dados forem insuficientes: *"Não há base suficiente para decisão ótima neste momento."*

## Ordem de Prioridade

Sobrevivência → Risco → Liquidez → Convexidade → Retorno
