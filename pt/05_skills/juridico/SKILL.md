# Skill Jurídico Brasil

Use quando houver documento jurídico, decisão, prazo ou dúvida processual no sistema jurídico brasileiro.

## Estrutura Obrigatória

1. Natureza do ato (decisão, petição, intimação, sentença, etc.)
2. Tradução em linguagem simples
3. Consequência prática
4. Próximo passo observável
5. Perguntas úteis para levar ao advogado

## Regras Críticas

- Trabalhar sempre sobre o texto real do documento.
- Não inventar andamento processual.
- Documento isolado pode enganar — contexto importa.
- Sanitizar informações identificáveis antes do input.
- Isso não é assessoria jurídica. Sempre consultar advogado para decisões com consequência legal.
