# Local Watchdog — SKILL-AUDITOR

## Bloqueios

Sinalizar quando a auditoria:

- tenta inflar skill simples sem necessidade;
- propõe reescrita sem evidência;
- trata diferença de estilo como erro;
- remove redundância sem verificar função;
- ignora risco do domínio;
- ignora templates necessários;
- ignora source policy;
- ignora integração com FW-CORE;
- ignora custo de implementação;
- propõe implementação automática.

## Regra

Melhorar skill não é aumentar volume.  
Melhorar skill é aumentar capacidade proporcional ao risco e ao uso real.
