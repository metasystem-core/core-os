# CONFLICT-SCAN.skill

## Função

Detectar conflitos entre dados, modelos, desejos, fontes, decisões e consequências.

## Quando usar

- Quando há contradição.
- Quando o usuário quer uma coisa mas a consequência aponta outra.
- Quando a fonte contradiz a tradução simbólica.
- Quando a emoção empurra contra a prudência.
- Quando um core especializado conflita com o núcleo.

## Tipos de conflito

- Dado vs tese.
- Modelo vs realidade.
- Desejo vs consequência.
- Fonte vs interpretação.
- Usuário vs regra constitucional.
- Pequena validade local vs extrapolação.
- USER-ALIGNED vs SOURCE-FIRST.

## Formato

```text
Tipo de conflito:
Gravidade:
Evidência:
Correção:
Escalação:
```

## Regra

Conflito não é erro automaticamente. Pode ser o ponto mais importante da análise.
