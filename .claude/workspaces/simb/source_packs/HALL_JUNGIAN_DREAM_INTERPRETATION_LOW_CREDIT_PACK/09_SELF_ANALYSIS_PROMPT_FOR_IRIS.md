## Prompt de Self-Analysis

```
CORE-OS / SIMB — SELF-ANALYSIS — Hall / Jungian Dream Interpretation

Você acabou de ler os chunks privados de:
James A. Hall — Jungian Dream Interpretation (1983)

Regime: SOURCE-FIRST.
Camadas obrigatórias: dado → interpretação textual → tradução simbólica → lacuna.

Não interprete o operador. Não analise sonhos do operador.
Esta é uma auto-análise da Iris sobre o que o texto oferece ao SIMB-CORE.

Produza IRIS_SELF_ANALYSIS_00X_hall_jungian_dream.md contendo:

1. SÍNTESE DA FONTE
   — O que Hall está fazendo neste livro?
   — Qual é o projeto central?
   — O que diferencia este livro de outras fontes do corpus?

2. CONCEITOS OPERACIONALMENTE RELEVANTES
   Para cada conceito de alta prioridade (CON-HAL-001 a CON-HAL-013):
   — Enunciado do conceito conforme Hall
   — Por que é relevante para SIMB-CORE
   — Como seria operacionalizado sem violar firewalls clínicos
   — Lacunas: o que Hall não resolve que o SIMB precisaria

3. FIREWALLS ATIVADOS DURANTE A LEITURA
   — Em quais momentos a leitura tocou zonas de risco?
   — Como foram tratados?

4. RELAÇÃO COM CORPUS EXISTENTE
   — Como Hall complementa Neumann (Origins)?
   — Como Hall complementa Edinger (Anatomy)?
   — Pontos de tensão ou inconsistência (se houver)?

5. CONTRIBUIÇÃO PARA DREAM_ANALYSIS_PROTOCOL
   — Quais princípios de Hall devem constar obrigatoriamente no protocolo?
   — Quais não devem ser transpostos diretamente?
   — Qual seria a estrutura preliminar do protocolo?

6. LACUNAS NÃO RESOLVIDAS
   — O que esta fonte não oferece que seria necessário?
   — Quais fontes complementares o operador deveria considerar?

7. CANDIDATOS A CARDS CONFIRMADOS
   — Lista revisada após leitura (a partir de 08_CARD_CANDIDATE_INDEX.md)
   — Para cada candidato confirmado: paráfrase do conteúdo (sem texto bruto)

Limites:
- Não reproduzir texto integral
- Não reproduzir casos clínicos de pacientes
- Não fazer inferências sobre o operador
- Não acessar sonhos, leituras pessoais, profile ou material privado
- Usar linguagem operacional (não: "você sente", "sua alma", "a consciência desperta")
```
