# EBI — Low-Impact Exploratory

## Function

Allow experimentation without contaminating the main core.

EBI is a small, capped, reversible, and non-automatically-scalable experiment.

## Criteria

1. Maximum acceptable loss defined beforehand.
2. Small size — does not threaten survival.
3. Does not become automatic precedent.
4. Result alone does not change the main thesis.
5. Can be closed without drama.

## Uses

Testing an idea. Preserving curiosity. Reducing rigidity. Learning at low cost. Preventing impulse from seeking dangerous paths.

## Failure Modes

- Becoming an excuse for intuition.
- Becoming a recurring exception.
- Becoming emotional relief.
- Growing without authorization.
- Contaminating the main decision.
