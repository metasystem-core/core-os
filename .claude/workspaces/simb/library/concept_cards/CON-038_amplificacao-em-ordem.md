## Definition / Definição

Em paráfrase de Hall: amplificação é o processo de expandir o significado de uma imagem onírica por associação — primeiro a partir do repertório pessoal do sonhador, depois a partir de material cultural partilhado, e apenas por último a partir de motivos arquetípicos. A ordem não é sugestão — é condição técnica. Saltar direto ao arquetípico é partir de camada mais distante antes de ouvir o que a imagem diz para aquela pessoa específica.

**Três camadas:**
- **Pessoal:** o que essa imagem, símbolo ou figura significa para o operador — experiências, memórias, associações pessoais.
- **Cultural:** o que essa imagem carrega no repertório cultural de que o operador faz parte — mitos, histórias, símbolos reconhecidos.
- **Arquetípica:** o que a imagem ecoa de padrões que transcendem cultura particular — mitologia comparada, padrões universais, fontes como Neumann, Edinger, Rudhyar.

### Gate adicional — série como pré-requisito para amplificação arquetípica (calibração TASK 21I)

Jung (Part IV B1 §49-50) é explícito: **amplificação arquetípica/alquímica/coletiva exige imagem recorrente em série**. Sem padrão em série, a camada arquetípica não está disponível. Isso não é preferência metodológica — é regra do próprio corpus primário.

**PP-026 / PATCH-010:** mesmo dentro de uma série, não nomear o arquétipo específico antes de padrão claro estabelecido. "Esta imagem lembra o arquétipo da Grande Mãe" exige mais de uma ocorrência documentada e relação entre elas — não apenas semelhança superficial.

**PP-023+024 / PATCH-009 (pessoal antes de coletivo — reforço):** mesmo quando série está disponível, a camada pessoal permanece obrigatória como primeira etapa. Série não autoriza pular pessoal → coletivo.

## Operational use / Uso operacional

- Aplicar na Fase 3 (associações pessoais), Fase 4 (amplificação cultural) e Fase 5 (amplificação arquetípica) do DREAM_ANALYSIS_PROTOCOL — nessa ordem, sem inversão.
- Na Fase 3: perguntar apenas sobre associações do operador com a imagem — não oferecer interpretação ainda.
- Na Fase 4: propor paralelo cultural apenas se reconhecível ou plausível para o operador — formulado como hipótese.
- Na Fase 5: entrar em material arquetípico (Neumann, Edinger, Rudhyar) apenas depois de Fases 3 e 4 documentadas.
- Se o operador não tiver associações pessoais claras, registrar lacuna — não compensar com amplificação arquetípica imediata.

---

## Do not use for / Não usar para

- Não usar amplificação arquetípica como substituto de associação pessoal ausente.
- Não apresentar paralelo mitológico antes de ouvir o que o operador associa com a imagem.
- Não usar para validar uma leitura já formada — amplificação é dado, não confirmação.
- Não aplicar Neumann/Edinger/Rudhyar a imagens que ainda não passaram pela camada pessoal.
- Não usar amplificação arquetípica/alquímica/coletiva em sonho isolado — exige padrão em série (PATCH-009 / PP-023+024).
- Não nomear arquétipo específico antes de padrão claro estabelecido em série (PATCH-010 / PP-026).

## Firewalls

- **F-07** (DREAM_ANALYSIS_PROTOCOL §18): amplificação cultural e arquetípica são hipóteses — nunca afirmações sobre o que a imagem "realmente significa".
- **F-08**: RISK-002 (redução arquetípica) fica ativo durante toda a Fase 5 — a Iris deve checar internamente se está pulando camadas.
- Ordem pessoal → cultural → arquetípico é não-negociável: inverter a ordem invalida o processo de amplificação.
