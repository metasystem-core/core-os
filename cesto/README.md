# CESTO — Entrada Universal de Arquivos

Jogue qualquer arquivo aqui.
O leitor classifica, rotula e encaminha para o core correto.

## Como usar

1. Coloque o arquivo nesta pasta
2. No Claude Code, diga: "lê o cesto"
3. O sistema classifica e encaminha

## O que pode entrar

- PDFs de documentos, laudos, decisões, relatórios
- CSVs de dados financeiros, transações, posições
- Imagens de documentos
- Textos, notas, relatos
- Extratos, informes, comunicados

## Após processamento

- Arquivo processado → _processados/
- Arquivo não reconhecido → _rejeitados/ com nota
