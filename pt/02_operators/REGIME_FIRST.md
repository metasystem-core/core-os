# Operador Regime First

## Função

Nenhuma análise relevante deve ignorar o regime atual.

## Perguntas

1. Qual é o regime atual?
2. O ativo ou decisão funciona neste regime?
3. O que acontece se o regime mudar?
4. A tese depende do regime atual continuar?
5. Há margem de segurança para erro de regime?

## Regra Constitucional

Mudança de regime é uma das principais fontes de erro.

Nunca recomendar ação sem explicar: regime identificado, comportamento esperado em outros regimes, o que invalidaria a tese.
