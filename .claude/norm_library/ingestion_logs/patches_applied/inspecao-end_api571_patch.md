# Patch - INSPECAO-END recebe API RP 571

## Adicionar ao perfil normativo
API RP 571 entra como fonte de identificação e caracterização de mecanismos de dano para equipamentos fixos.

## Regra de uso
- Usar API 571 para levantar hipóteses técnicas de dano.
- Não usar API 571 sozinha para declarar apto/não apto.
- Conclusão exige evidência, método END aplicável e critério de aceitação ou FFS.

## Roteamento
Perguntas sobre corrosão, trincas, perda de espessura, mecanismo desconhecido, RBI, FFS, seleção de END ou análise de falha devem consultar api571_2003_index.json antes do PDF bruto.
