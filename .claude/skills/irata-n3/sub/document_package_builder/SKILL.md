# SUBSKILL — Document Package Builder

## Função

Montar pacote documental para uma frente de acesso por cordas.

## Pacote típico

- procedimento de acesso;
- plano de resgate;
- plano de ancoragem;
- APR/PT/AF;
- matriz de equipe;
- matriz de equipamentos;
- certificados/rastreabilidade;
- checklist de liberação;
- croquis/fotos;
- relatório de lacunas;
- registro de comunicação com cliente.

## Saída

Lista de documentos:

- disponível;
- pendente;
- não aplicável;
- precisa validação;
- bloqueio.
