# Política — SKILL.md não é banco de dados

## Regra

SKILL.md contém:
- função;
- gatilhos;
- limites;
- fluxo;
- regras;
- interfaces.

SKILL.md não deve conter:
- blocos normativos;
- dados extensos;
- registros de carteira;
- histórico documental;
- listas longas que dependem do filesystem;
- estado projetado sem registry.

## Fonte de verdade

- registry = estado real;
- library = arquivos/conteúdo;
- index = navegação;
- blocks = conteúdo refinado;
- SKILL.md = instrução operacional.
