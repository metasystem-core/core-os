# SKILL — Core Versioning Protocol

## Função

Controlar evolução estrutural do CORE-OS.

## Quando usar

- nova regra constitucional;
- alteração de arquitetura;
- criação de core;
- criação de camada transversal;
- obsolescência de skill;
- correção de erro importante;
- migração para outra IA.

```markdown
- incorporada
- rejeitada
- obsoleta
```

## Regra

Sistema vivo sem versionamento vira memória confusa.
