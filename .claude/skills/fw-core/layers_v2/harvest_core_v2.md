# SKILL — HARVEST-CORE / Elevação de Aprendizados

## Função

Varredura de aprendizados nos cores especializados para propor elevação ao NÚCLEO.

Os cores especializados não alteram o FW-CORE diretamente. Eles sugerem. O NÚCLEO decide.

## Critérios para subir

Subir quando afeta múltiplos domínios, corrige erro recorrente, protege contra risco relevante ou melhora a arquitetura.
