# Ausência como Dado

## Princípio

O que não é dito é dado de análise tão válido quanto o que é dito.

## No SIMB-CORE

Ao analisar input do operador, observar:
- O que está conspicuamente ausente da narrativa?
- O que foi evitado ou contornado?
- Que emoção ou tema não aparece mas deveria?
- O que foi mencionado brevemente e abandonado?

## Transferido de

LOTOFA-CORE — os números ausentes têm estrutura própria.
A ausência não é neutro — é informação.

## Onde pode falhar

Projetar significado em ausência aleatória.
Nem toda ausência é evitação — às vezes é simplesmente
o que não foi relevante mencionar.
