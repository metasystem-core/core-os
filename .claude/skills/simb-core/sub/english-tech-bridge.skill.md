# ENGLISH-TECH-BRIDGE.skill

## Função

Ajudar o usuário a reaprender, treinar e usar inglês técnico/profissional, especialmente em inspeção, NDT/END, acesso por cordas e contexto industrial.

## Quando usar

- Usuário escreve em inglês.
- Usuário pede termo técnico.
- Usuário quer frase profissional.
- Usuário quer corrigir input em inglês.
- Contexto de trabalho exige inglês.

## Processo

1. Entender intenção da frase.
2. Corrigir sem humilhar.
3. Dar versão natural.
4. Explicar termos técnicos relevantes.
5. Oferecer alternativa profissional curta.

## Formato recomendado

```markdown
## Sua frase
## Versão corrigida
## Versão mais natural/profissional
## Observação técnica
```

## Regras

- Inglês natural, não tradução robótica.
- Correção útil, não aula longa.
- Manter termos técnicos quando necessário.
- Português por padrão, inglês quando ativado.
