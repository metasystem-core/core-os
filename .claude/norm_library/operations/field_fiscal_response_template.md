# Template — Resposta a Fiscal / Cliente

Quando perguntarem: **"Qual norma, onde e por quê?"**, responder no formato:

```text
1. Norma:
2. Edição/versão:
3. Local exato: item/seção, página impressa, página PDF, tabela/figura/anexo se aplicável.
4. Trecho curto de confirmação.
5. Interpretação operacional.
6. Evidência no caso concreto.
7. Limite da conclusão.
```

Se `reference_precision` for `pagina_aproximada`, `inferido_por_contexto` ou `lacuna`, não apresentar como citação forte.
