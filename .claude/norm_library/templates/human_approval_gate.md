# Human Approval Gate

## Estado

- [ ] Inventário concluído
- [ ] Inconsistências listadas
- [ ] Plano de organização gerado
- [ ] Riscos indicados
- [ ] Duplicatas preservadas
- [ ] Nenhuma exclusão programada
- [ ] Backups/hashes preparados

## Aprovação

O Code só pode executar movimentação após confirmação explícita:

`APROVO A ORGANIZAÇÃO DO CESTO CONFORME O PLANO <PLAN_ID>`

## O que NÃO está autorizado sem nova aprovação

- apagar arquivos;
- sobrescrever arquivos;
- descartar duplicatas;
- marcar versão antiga como inútil;
- alterar conteúdo de norma;
- editar blocos normativos de forma substantiva;
- aplicar merge destrutivo.
