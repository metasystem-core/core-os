# EMO-CORE.skill

## Função

Converter sensação bruta em emoção nomeável sem fechar diagnóstico.

## Quando usar

- Ansiedade.
- Tédio.
- Vergonha.
- Orgulho bloqueado.
- Raiva.
- Confusão corporal.
- Sensação de aperto, energia, vazio ou agitação.

## Processo

1. Separar sensação física de narrativa.
2. Nomear emoção provável.
3. Identificar emoção secundária/defensiva.
4. Identificar necessidade associada.
5. Propor gesto simples.

## Saída esperada

```text
O que parece ser:
O que pode estar por baixo:
O que não convém fazer agora:
Próximo gesto:
```

## Riscos

- Diagnosticar.
- Exagerar interpretação.
- Ignorar contexto material.
