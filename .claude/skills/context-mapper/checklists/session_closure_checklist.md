# Checklist de fechamento de sessão

## 1. Identificação

- [ ] Data da sessão registrada?
- [ ] Projeto/core dominante identificado?
- [ ] Cores secundários identificados?
- [ ] Objetivo da sessão resumido?

## 2. Decisões

- [ ] Decisões tomadas foram extraídas?
- [ ] Decisões estão separadas de ideias?
- [ ] Decisões estruturais têm versioning note?
- [ ] O que ainda não foi decidido está claro?

## 3. Ideias novas

- [ ] Ideias novas foram listadas?
- [ ] Ideias estão marcadas como propostas, não como regras?
- [ ] Ideias candidatas a skill foram separadas?
- [ ] Ideias candidatas a código foram separadas?

## 4. Arquivos e artefatos

- [ ] ZIPs gerados listados?
- [ ] All-in-one listados?
- [ ] Caminhos dos arquivos registrados?
- [ ] Função de cada arquivo registrada?
- [ ] Próximo uso de cada artefato definido?

## 5. Pendências

- [ ] Tarefas do operador listadas?
- [ ] Tarefas para Claude listadas?
- [ ] Tarefas para ChatGPT listadas?
- [ ] Tarefas de runtime offline listadas?
- [ ] Prioridade definida?

## 6. Riscos/lacunas

- [ ] Riscos detectados?
- [ ] Lacunas abertas?
- [ ] Risco de custo/Claude?
- [ ] Risco de privacidade?
- [ ] Risco de arquitetura?

## 7. Memória/HARVEST

- [ ] Candidatos a memória durável listados?
- [ ] Candidatos a HARVEST-CORE listados?
- [ ] Nada foi incorporado automaticamente?

## 8. Ruído

- [ ] O que não preservar foi declarado?
- [ ] Digressões foram removidas?
- [ ] Repetições foram compactadas?

## 9. Próximo ciclo

- [ ] Próxima ação mínima definida?
- [ ] Prompt/contexto para próxima sessão preparado?
- [ ] Prompt para Claude, se necessário, preparado?
