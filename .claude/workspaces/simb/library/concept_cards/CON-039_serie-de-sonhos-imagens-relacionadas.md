## Definition / Definição

Em paráfrase de Hall: uma série de sonhos não exige que a mesma imagem reapareça — imagens distintas podem ser expressões variadas de um mesmo campo em transformação. O que constitui série é relação de campo, não repetição literal. A variação entre sonhos de uma mesma série é dado tão importante quanto a recorrência: indica como o campo se move, não apenas que ele persiste.

Recorrência literal (mesma imagem, mesmo cenário) é um subconjunto de série — não o caso geral.

---

## Operational use / Uso operacional

- Uma série de sonhos é mais informativa do que um sonho isolado porque permite observar transformação, repetição, deslocamento de imagem, mudança de afeto e mudança de ação do dream-ego ao longo do tempo. Mesmo assim, série não autoriza conclusão definitiva — ela melhora a qualidade da leitura provisória, não a encerra.
- Na Fase 7 (DREAM_ANALYSIS_PROTOCOL §11): usar este conceito para avaliar se o sonho atual pertence a uma série — verificar DREAM_REGISTRY antes de concluir.
- Quando o operador verbaliza "essa imagem voltou" ou "é de novo aquele lugar": registrar como `[recorrência verbalizada / não indexada]` se não houver histórico formal — não tratar como série confirmada.
- Pedir autorização para iniciar rastreamento no DREAM_REGISTRY quando série parece emergir.
- Ler variação entre sonhos de uma série como dado de transformação — não como inconsistência.
- Não concluir padrão de série a partir de sonho isolado, mesmo que o operador verbalize familiaridade.

---

## Do not use for / Não usar para

- Não usar para afirmar que uma imagem tem significado fixo por aparecer mais de uma vez.
- Não usar recorrência como prova de que "isso é muito importante" sem base no DREAM_REGISTRY.
- Não usar variações dentro de série como argumento de que "o campo piorou" ou "melhorou" — é dado de movimento, não de julgamento.
- Não construir narrativa linear de "progresso" a partir de série — série não é arco de melhora.

## Firewalls

- **F-03** (DREAM_ANALYSIS_PROTOCOL §18): série não exclui provisoriedade — leitura de série também é hipótese.
- `[recorrência verbalizada / não indexada]` é marcação obrigatória quando operador alega recorrência sem histórico formal — nunca tratar verbalizações como série confirmada.
- Padrão definitivo de série exige histórico no DREAM_REGISTRY — sem registro, toda conclusão de série é especulativa.
