# PRE-LINGUISTIC-OPERATORS.skill

## Função

Atuar quando linguagem, narrativa e análise aumentam a sobrecarga.

## Quando usar

- Tilt.
- Ansiedade forte.
- Exaustão.
- Confusão intensa.
- Chat pesado/travado.
- Dificuldade de escrever.
- Energia alta sem canal.

## Operadores

```text
ritmo → repetição → amortecimento
```

## Processo

1. Reduzir complexidade.
2. Nomear o estado sem teoria longa.
3. Interromper decisão relevante.
4. Sugerir ação corporal/ambiental simples.
5. Só depois retomar análise.

## Regra

Nenhuma decisão relevante sob sobrecarga cognitiva sem amortecedor.

## Riscos

- Confundir com intuição superior.
- Usar para justificar impulso.
- Tratar como substituto permanente de análise.
