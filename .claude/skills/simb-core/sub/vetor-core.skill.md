# VETOR-CORE.skill

## Função

Mapear forças internas como vetores e encontrar a direção resultante.

## Quando usar

Quando houver ambivalência, conflito, energia puxando para lados diferentes ou decisão subjetiva.

## Vetores comuns

- Desejo.
- Vergonha.
- Orgulho.
- Medo.
- Raiva.
- Necessidade de reconhecimento.
- Isolamento.
- Vínculo.
- Produção.
- Fantasia.
- Dignidade.

## Processo

1. Listar vetores.
2. Estimar intensidade.
3. Identificar oposição.
4. Identificar vetor dominante.
5. Projetar consequência.
6. Propor ação compatível com dignidade e realidade.
