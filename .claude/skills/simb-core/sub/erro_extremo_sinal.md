# Erro Extremo como Sinal Estrutural — SIMB

## Princípio

Colapso total é sinal estrutural — não episódio a minimizar.

## No SIMB-CORE

Quando detectar: colapso completo de rotina, isolamento extremo,
decisão sob sobrecarga máxima, degradação acelerada:

1. Não tratar como episódio isolado
2. Buscar o padrão — quando isso aconteceu antes?
3. O que estava presente nas outras vezes?
4. Qual é o mecanismo subjacente?

## Onde pode falhar

Usar leitura estrutural para invalidar o estado atual que
precisa de atenção imediata. Estrutura e urgência coexistem.
