# SUBSKILL — Normative Reference Router

## Função

Identificar quais fontes normativas/documentais devem ser consideradas para a atividade.

## Fontes possíveis

- NR-35;
- IRATA ICOP;
- TACS/documentos IRATA aplicáveis;
- NBRs aplicáveis;
- procedimento interno do cliente;
- procedimento interno da empresa;
- APR/PT/AF;
- manual do fabricante;
- documentação de engenharia;
- plano específico da frente.

## Regra

Se a vigência da norma/procedimento não estiver confirmada, declarar como lacuna antes de usar como base final.
