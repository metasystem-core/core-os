# Checklist de classificação de lacuna

## Demanda

- [ ] Qual foi a demanda original?
- [ ] Qual core/skill tentou responder?
- [ ] Onde travou?
- [ ] Qual capacidade era esperada?
- [ ] Qual capacidade existe hoje?

## Tipo de lacuna

- [ ] skill_gap
- [ ] skill_weakness_gap
- [ ] template_gap
- [ ] runtime_code_gap
- [ ] data_source_gap
- [ ] routing_gap
- [ ] safety_policy_gap
- [ ] memory_gap
- [ ] schema_gap
- [ ] qa_gap

## Gravidade

- [ ] baixa
- [ ] média
- [ ] alta
- [ ] crítica

## Recorrência

- [ ] única
- [ ] ocasional
- [ ] recorrente
- [ ] estrutural

## Solução recomendada

- [ ] criar skill
- [ ] adaptar skill
- [ ] criar subskill
- [ ] criar template
- [ ] criar script/runtime
- [ ] atualizar router
- [ ] atualizar source policy
- [ ] criar watchdog
- [ ] bloquear/escala
- [ ] não criar nada
