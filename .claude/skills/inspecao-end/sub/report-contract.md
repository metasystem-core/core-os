# Sub-skill: Contrato de Laudo

Consultada quando: verificar conteúdo mínimo de laudo ou estruturar novo.

## Conteúdo mínimo obrigatório

### Identificação
- [ ] TAG do equipamento
- [ ] Descrição e função
- [ ] Localização (planta, área)
- [ ] Fluido de serviço
- [ ] Parâmetros de projeto (pressão, temperatura, material)

### Objetivo e escopo
- [ ] Objetivo da inspeção
- [ ] Escopo inspecionado
- [ ] O que NÃO foi inspecionado e por quê

### Metodologia
- [ ] Técnica(s) END utilizadas
- [ ] Procedimento de referência (número, revisão)
- [ ] Equipamento utilizado (modelo, série, calibração válida)
- [ ] Qualificação do executor (nível, certificado, validade)

### Resultados
- [ ] Descrição de cada achado (localização, dimensão, orientação)
- [ ] Evidência fotográfica
- [ ] Registro dimensional

### Avaliação
- [ ] Critério de aceitação (norma, cláusula)
- [ ] Classificação de cada achado
- [ ] Mecanismo de dano provável

### Conclusão
- [ ] Situação do equipamento
- [ ] Recomendação
- [ ] Prazo para próxima inspeção

### Responsabilidade
- [ ] Nome, registro profissional, qualificação END, data, assinatura
