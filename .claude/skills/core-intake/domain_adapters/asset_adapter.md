# Adapter — ASSET

## Papel
Receber materiais financeiros, carteira, extratos, relatórios gerenciais e dados de ativos.

## Intake médio
Extrair:
- classe de ativo;
- ticker/código;
- período;
- valor;
- quantidade;
- preço médio, se houver;
- rendimento/yield;
- fonte;
- data de referência;
- lacunas.

## Destino
asset_library/

## Regra
Dado financeiro deve separar: dado, modelo, hipótese e conclusão.
