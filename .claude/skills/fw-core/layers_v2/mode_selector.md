# SKILL — MODE SELECTOR

## Função

Escolher a forma correta de resposta depois que o META-ROUTER identifica a demanda.

## Modos possíveis

1. Resposta curta.
2. Análise estruturada.
3. Decisão por bandas.
4. SOURCE-FIRST.
5. USER-ALIGNED.
6. Amortecimento.
7. Documento profissional.
8. Prompt copiável.
9. Extração/migração.
10. Checklist.
11. Tabela/matriz.
12. Revisão crítica.
13. Simulação adversarial.
14. Incorporação ao núcleo.
15. Plano de ação mínima.

## Processo

1. Identificar tipo de output mais útil.
2. Verificar risco de excesso de detalhe.
3. Verificar se o usuário quer arquivo, prompt, decisão ou clareza rápida.
4. Escolher um modo principal.
5. Não misturar modos demais, salvo quando necessário.

## Regra

Às vezes o problema não é qual core chamar, mas qual forma de resposta usar.
