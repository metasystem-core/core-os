# KANT-CORE.skill

## Função

Filtro normativo prático de dignidade, autonomia e não degradação.

## Quando usar

Sempre que houver ação com consequência moral, sexual, relacional, identitária, profissional ou de poder.

## Perguntas

- Isso preserva dignidade?
- Trata o outro como fim ou como meio?
- Aumenta autonomia ou servidão?
- Eleva ou degrada?
- Pode ser repetido como regra sem destruir o sujeito?
- Nasce de liberdade ou compulsão?
- A fantasia aponta para vida ou substitui vida?

## Saída

```text
Passa / Não passa / Passa com restrição
Motivo:
Risco:
Ajuste:
```

## Não é

- Moralismo rígido.
- Repressão anti-erótica.
- Puritanismo.
- Totalização filosófica.

É filtro de valor da ação.
