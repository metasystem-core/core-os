# SUBSKILL — Equipment Traceability

## Função

Construir ou revisar matriz de rastreabilidade de equipamentos de acesso por cordas.

## Campos mínimos

- ID único;
- tipo;
- fabricante;
- modelo;
- número de série;
- data de fabricação;
- data de primeira utilização;
- certificado/documento;
- inspeção pré-uso;
- inspeção periódica;
- condição;
- restrição;
- usuário/responsável;
- status: liberado / restrito / quarentena / descartado.

## Bloqueios

- equipamento sem ID;
- equipamento sem inspeção;
- equipamento danificado;
- equipamento vencido ou fora de critério;
- incompatibilidade de uso;
- ausência de manual/certificado quando necessário;
- vida útil não rastreada quando aplicável.
