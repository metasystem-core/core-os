# SKILL — E-Histórico Engine

## Função

Estudar erros, perdas, falhas e estados adversos antes de buscar ganho.

## Definição

E = conjunto de erros, perdas, drawdowns, falhas, ausências, resultados negativos ou estados adversos possíveis.

## Perguntas obrigatórias

- Qual é o E dominante?
- Qual é o pior caso plausível?
- O sistema sobrevive?
- Qual é o custo de sobreviver?
- O erro é reversível?
- O erro é raro, mas fatal?
- O erro é comum, mas absorvível?
- O erro muda por regime?
- O erro já apareceu antes?

## Tradução por domínio

### Investimentos
Drawdown, iliquidez, perda permanente, mudança de regime, erro de tese.

### BTC
Volatilidade extrema, macro hostil, convicção identitária, compressão de liquidez.

### Lotofácil
Números ausentes, erro combinatório, padrão falso, overfitting.

### SIMB
Ruminação, fantasia substituindo ação, degradação, isolamento.

### IRATA/Inspeção
Acidente, falha de ancoragem, resgate inviável, equipamento incompatível.

### Jurídico
Prazo perdido, interpretação errada, falsa certeza, consequência processual mal lida.

## Regra

Antes de perguntar “quanto posso ganhar?”, perguntar “como isso pode me quebrar?”.
