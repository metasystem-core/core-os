# SUBSKILL — Team Competency Matrix

## Função

Mapear equipe, certificações, funções e restrições.

## Campos

- nome;
- função na atividade;
- certificação IRATA;
- validade;
- treinamentos aplicáveis;
- ASO/aptidão, se aplicável;
- função no resgate;
- restrições;
- observações.

## Bloqueios

- ausência de N3 quando necessário;
- certificação vencida;
- equipe sem competência para resgate;
- função crítica sem responsável definido;
- atividade incompatível com competência disponível.
