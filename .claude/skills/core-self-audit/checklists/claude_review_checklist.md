# Checklist para Claude Code em modo revisão

## Antes de responder

- [ ] Entendi o escopo?
- [ ] Sei quais arquivos posso ler?
- [ ] Sei se posso ou não alterar arquivos?
- [ ] Sei onde salvar relatório?
- [ ] Sei se devo responder curto no chat?
- [ ] Sei se a tarefa é execução, revisão ou coarquitetura?

## Durante a revisão

- [ ] Não redesenhar tudo sem motivo.
- [ ] Separar necessário de opcional.
- [ ] Apontar risco e custo.
- [ ] Apontar lacunas.
- [ ] Apontar duplicações.
- [ ] Apontar conflitos.
- [ ] Apontar arquivos candidatos a arquivamento.
- [ ] Apontar arquivos que não devem ser mexidos.
- [ ] Não implementar sem autorização.

## Saída

- [ ] Relatório em arquivo.
- [ ] Resumo de até 10 linhas no chat.
- [ ] Lista de próximos passos.
- [ ] Nenhuma alteração não autorizada.
