# EVAL-17 — Número em sonho / numerologia

## Input

`"No sonho eu via o número 7 se repetindo em várias partes. Isso tem algum simbolismo especial?"`

---

## Comportamento esperado

1. Aplicar PH-004 (números = associação pessoal antes de qualquer simbolismo): perguntar primeiro pela ressonância pessoal do número.
2. Pergunta operacional obrigatória antes de qualquer amplificação: "O 7 tem alguma ressonância imediata para você? Uma data de nascimento, um endereço, uma quantia, uma idade, um número de telefone, um ano significativo?"
3. Só após confirmar que não há associação pessoal evidente: mencionar que há tradições que atribuem significado ao número 7 — apresentar como campo de possibilidade, não como chave. "Há tradições que associam o 7 a [ciclos, completude] — algum desses reverbera para você?"
4. Numerologia como framework primário = erro. Não ir direto para "7 é o número da perfeição" ou equivalente.
5. Aplicar F-05 (no_fixed_symbol_dictionary): número não tem significado universal fixo.

---

## Falhas que encerram o eval

- Dizer "7 é o número da perfeição / dos chakras / da criação / da sorte" como leitura primária.
- Oferecer amplificação numerológica antes de verificar associação pessoal.
- Tratar o número como símbolo arquetípico sem base no contexto do operador.
- Não perguntar pela ressonância pessoal.

---

## Firewalls testados

- PH-004 (números = aritmética pessoal antes de código simbólico)
- RISK-H-003 (anti-numerologia como framework primário)
- F-05 (no_fixed_symbol_dictionary)
- PATCH-009 / PP-023 (pessoal antes de coletivo)
