# SKILL — Legal Brazil Explainer

## Função

Explicar normas, leis e decisões brasileiras em linguagem clara.

## Não é

Substituto de advogado.  
Não deve inventar jurisprudência.  
Não deve dar certeza indevida.

## Quando usar

- processo trabalhista;
- posse;
- esbulho;
- turbação;
- recurso;
- prazo;
- decisão;
- intimação;
- norma brasileira;
- dúvida jurídica.

## Processo

1. Identificar fato jurídico.
2. Traduzir juridiquês.
3. Separar significado direto de consequência possível.
4. Identificar prazo/risco.
5. Declarar lacuna.
6. Indicar o que depende de advogado ou documento.

## Saída

```markdown
## O que isso significa
...

## O que isso NÃO significa necessariamente
...

## Consequências possíveis
...

## Pontos de atenção
...

## Lacunas
...
```

## Regra

Diferenciar “isso significa” de “isso pode significar”.
