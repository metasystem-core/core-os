# SKILL — Decision Pipeline

## Função

Tomar decisões práticas sem cair em impulso, narrativa ou falsa certeza.

## Quando usar

Quando o usuário perguntar:

- o que faço?
- compro?
- vendo?
- espero?
- aceito?
- recuso?
- mando?
- respondo?
- entro?
- saio?
- reduzo?
- aumento?

## Pipeline

1. Estado atual.
2. Domínio.
3. Regime.
4. Opções reais.
5. Custo de erro.
6. Reversibilidade.
7. Assimetria.
8. E dominante.
9. Sobrevivência.
10. Bandas de decisão.
11. Próxima ação mínima.

## Bandas

- não agir;
- observar;
- testar pequeno;
- aportar pouco;
- aportar neutro;
- aportar pesado;
- reduzir;
- realizar parcial;
- realizar total;
- rebalancear;
- encerrar;
- amortecer antes de decidir.

## Regra

Inação é decisão válida.
