# HARVEST-CORE.skill

## Função

Avaliar se um aprendizado da conversa deve virar memória estrutural, regra, operador ou subskill.

## Quando usar

Quando surgir:

- formulação nova;
- regra operacional;
- insight recorrente;
- módulo possível;
- erro corrigido;
- distinção útil;
- padrão do usuário;
- regra de produto.

## Processo

```text
Origem:
Aprendizado:
Tipo:
Motivo:
Escopo:
Não-escopo:
Risco de transposição:
Cores afetados:
Recomendação: descartar / guardar / testar / constitucionalizar
```

## Riscos

- Guardar demais.
- Constitucionalizar cedo demais.
- Transformar experiência pessoal em regra universal.
