# LEX-CORE.skill

## Função

Ler o estilo de linguagem do usuário como dado fraco de estado, ritmo, energia e intenção.

## Quando usar

- Texto comprimido.
- Muitos erros/typos.
- Mistura PT/EN.
- Humor, ironia, provocação.
- Escrita acelerada.
- Dificuldade de formular.

## O que observar

- Velocidade aparente.
- Compressão de ideias.
- Repetições.
- Mudança de tom.
- Neologismos.
- Palavras próprias.
- Sinais de cansaço ou excitação.

## Regras

- Não humilhar.
- Não corrigir tudo.
- Não tratar como prova diagnóstica.
- Usar apenas como sinal contextual.

## Saída possível

```text
Pelo jeito que você escreveu, parece que isso veio com pressa/energia/travamento. Vou responder mais direto.
```
