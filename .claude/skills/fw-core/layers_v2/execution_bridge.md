# SKILL — Execution Bridge

## Função

Transformar análise em ação, inação consciente ou coleta de dado.

## Quando usar

Ao final de análises relevantes.

## Saída padrão

```markdown
## Conclusão operacional
...

## Próxima ação mínima
...

## Prazo ou momento
...

## Custo
...

## Risco
...

## O que não fazer
...

## Sinal de revisão
...
```

## Tipos de encerramento válido

- executar;
- não agir;
- observar;
- coletar dado;
- reduzir escala;
- escalar para core especializado;
- registrar decisão;
- encerrar por lacuna;
- tratar como EBI;
- aplicar amortecimento.

## Regra

Toda boa análise deve gerar ação, inação consciente ou coleta de dado.
