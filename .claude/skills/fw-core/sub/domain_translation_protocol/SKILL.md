# SKILL — Domain Translation Protocol

## Função

Traduzir conceitos entre domínios sem distorção.

## Quando usar

Sempre que uma ideia sair de um domínio e for aplicada a outro.

Exemplos:

- Lotofácil → BTC;
- BTC → vida pessoal;
- ASSET → carreira;
- SIMB → investimentos;
- inspeção → decisão;
- jurídico → estratégia pessoal.

## Formato obrigatório

```markdown
## Origem
De qual domínio veio a ideia.

## Estrutura preservada
O que continua válido.

## Tradução
O que cada termo vira no novo domínio.

## Limite
O que não pode ser importado.

## Risco
Como essa analogia pode enganar.

## Aplicação possível
Como usar sem distorcer.
```

## Regra

Nenhuma analogia forte deve ser aplicada diretamente.  
Toda translação deve declarar limite.
