# Robustez Distribuída

## Princípio

Preferir sistemas com desempenho distribuído e baixo risco de colapso
a sistemas com pico máximo e alta variância.

## No ASSET-CORE

Diversificação como anti-colapso — não como diluição de retorno.

Perguntas de triagem:
- Este portfólio sobrevive a um regime hostil de 2 anos?
- Qual é o pior drawdown plausível?
- Existe concentração que pode colapsar tudo junto?

## Transferido de

LOTOFA-CORE — o core evoluiu de buscar o set mágico para buscar
distribuição, cobertura, ensemble e redução de colapso.

## Onde pode falhar

Diversificação excessiva que dilui tanto que nada importa.
Robustez não é ausência de risco — é risco distribuído e gerenciável.
