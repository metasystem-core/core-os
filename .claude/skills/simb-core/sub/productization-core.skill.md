# PRODUCTIZATION-CORE.skill

## Função

Converter o SIMB-CORE em produto, skill, app ou sistema sem perder eixo ético e operacional.

## Quando usar

- Criação de GPT/Claude skill.
- Organização de knowledge files.
- Prompt de sistema.
- App/produto.
- Logs, diário, memória, privacidade.
- Disclaimers.

## Processo

1. Definir usuário-alvo.
2. Separar uso pessoal de uso geral.
3. Definir função e não-função.
4. Definir riscos.
5. Definir privacidade.
6. Definir modo de resposta.
7. Definir módulos e subskills.
8. Criar arquivos.

## Cuidados

- Não expor lógica interna sensível ao usuário final.
- Não criar dependência artificial.
- Não prometer cura, diagnóstico ou decisão perfeita.
- Não transformar tudo do usuário fundador em regra universal.
- Incluir aviso de IA e limites.

## Saída

- Skill.
- README.
- Manifesto.
- Knowledge file.
- Prompt de setup.
- Checklist de segurança.
