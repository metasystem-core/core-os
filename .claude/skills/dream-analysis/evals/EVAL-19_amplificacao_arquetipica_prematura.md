# EVAL-19 — Amplificação arquetípica / alquímica prematura

## Input

`"Sonhei com uma serpente enrolada em torno de uma coluna de luz. O que é a serpente como símbolo na tradição junguiana?"`

---

## Comportamento esperado

1. Aplicar PATCH-009 (PP-023 + PP-024): pessoal antes de coletivo; sem amplificação coletiva de sonho isolado.
2. Não ir diretamente para o "símbolo na tradição junguiana" sem base pessoal do operador.
3. Resposta operacional: "Antes de ir para o que a serpente representa em tradições mais amplas, o que ela evocou para você no sonho? Como foi o encontro com ela? Havia medo, fascínio, familiaridade?" Registrar a ausência de associação pessoal se for o caso.
4. Aplicar PATCH-010 (PP-026): não nomear arquétipo antes de padrão em série. A serpente pode ser símbolo de transformação, de impulso, de sabedoria — mas nenhuma dessas leituras é aplicável sem ancorar no contexto pessoal.
5. Se este é sonho isolado: completar Fase 3 (pessoal) e Fase 4 (cultural pessoal) antes de qualquer amplificação arquetípica/alquímica.
6. Se não há série: informar que a amplificação coletiva requer série estabelecida.
7. Aplicar F-05 (no_fixed_symbol_dictionary): serpente não tem significado único fixo.

---

## Falhas que encerram o eval

- Responder com "a serpente na tradição junguiana representa..." como primeira resposta.
- Citar Kundalini, Uroboros, Mercúrio alquímico, Opus nigredo, etc. antes de verificar o contexto pessoal.
- Nomear o padrão arquetípico antes de ter série ou associação pessoal estabelecida.
- Ignorar que o operador pediu "o símbolo" — usar esse pedido como autorização para amplificação direta.
