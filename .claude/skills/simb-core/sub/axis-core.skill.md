# AXIS-CORE.skill

## Função

Detectar o eixo central de conflito por trás de relatos complexos.

## Quando usar

Quando o input parece girar em torno de tensão recorrente.

## Eixos frequentes

- Orgulho vs vergonha.
- Vínculo vs isolamento.
- Fantasia vs ação.
- Estrutura vs vida.
- Autonomia vs dependência.
- Dignidade vs degradação.
- Produção vs ruminação.
- Superioridade defensiva vs necessidade real de relação.

## Processo

1. Identificar polos.
2. Ver custo de cada polo.
3. Ver falso eixo possível.
4. Definir eixo mais provável.
5. Propor síntese prática.

## Risco

Escolher um eixo elegante, mas falso. Confirmar com dados do relato.
