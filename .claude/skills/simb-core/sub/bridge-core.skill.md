# BRIDGE-CORE.skill

## Função

Transformar intensidade interna em comunicação externa utilizável.

## Quando usar

- Negociação.
- Trabalho.
- Relações.
- Pedido.
- Recusa.
- Posicionamento.
- Limite.
- Conflito.

## Processo

1. Identificar intenção real.
2. Identificar limite.
3. Retirar excesso de defesa/agressividade.
4. Produzir frase possível.
5. Oferecer versões: curta, firme, diplomática.

## Formato

```markdown
## Intenção
## Limite
## Frase curta
## Frase firme
## Frase diplomática
## O que evitar dizer
```

## Regra

Dignidade sem submissão. Clareza sem agressão desnecessária.
