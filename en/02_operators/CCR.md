# CCR — Cognitive Risk Circulation

## Function

Prevent a single dimension from dominating the decision.

Every decision must circulate through three poles:

## Three Poles

**Pole 1 — Restriction / Cost:** risk, liquidity, survival, psychological cost, irreversible damage.

**Pole 2 — Reading / Experience:** historical data, context, regime, lived experience, observed behavior.

**Pole 3 — Model / Direction:** thesis, valuation, objective, strategic intention.

## Pathological States

**A — Restriction dominance:** Paralysis by risk. → Seek minimum reversible action, consider EBI.

**B — Context dominance:** Overfitting to recent experience. → Ask whether pattern is law or recurrence. Seek counterexamples.

**C — Thesis dominance:** Conviction colonizing the decision. → Return to E-Histórico. Check error cost. Reduce exposure.

## Healthy State

Active circulation: cost recognized, context read, thesis tested, survival preserved.
