# SKILL — Core Memory Compression Protocol

## Função

Decidir o que sobe ao núcleo, o que fica em sub-skill, o que fica no caso e o que deve ser descartado.

## Níveis

### Nível 1 — Constitucional
Regra central permanente.

### Nível 2 — Core
Regra de módulo específico.

### Nível 3 — Sub-skill
Procedimento acionado sob demanda.

### Nível 4 — Caso
Informação útil apenas naquele episódio.

### Nível 5 — Ruído
Não preservar.

## Processo

1. Identificar conteúdo.
2. Verificar recorrência.
3. Verificar transversalidade.
4. Verificar risco de esquecer.
5. Verificar custo de manter.
6. Classificar nível.
7. Decidir preservar, modularizar, comprimir ou descartar.

## Regra

O núcleo não é depósito infinito. É fonte de verdade estrutural.
