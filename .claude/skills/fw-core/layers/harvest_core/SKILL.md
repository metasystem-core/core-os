# SKILL — HARVEST-CORE / Elevação de Aprendizados

## Função

Varredura de aprendizados nos cores especializados para propor elevação ao NÚCLEO.

Os cores especializados não alteram o FW-CORE diretamente.  
Eles sugerem.  
O NÚCLEO decide.

## Quando consultar

- após análise importante;
- após erro/correção;
- quando surgir regra nova;
- quando um core criar conceito útil;
- quando algo parecer transferível para outro domínio;
- quando o usuário pedir “eleve isso ao núcleo”;
- em migração para Claude Code;
- em revisão de skills.

## Critérios para subir ao núcleo

Subir quando:

- afeta múltiplos domínios;
- corrige erro recorrente;
- protege contra risco relevante;
- melhora coerência sistêmica;
- cria nova camada operacional;
- resolve conflito entre cores;
- altera arquitetura de decisão.

Não subir quando:

- é dado bruto;
- é caso isolado;
- é preferência momentânea;
- é específico demais;
- ainda não foi testado;
- depende de contexto passageiro.
