# SKILL — Skill Lifecycle Protocol

## Função

Controlar o ciclo de vida das skills e evitar acúmulo de módulos obsoletos.

## Status possíveis

### Proposta
Ideia ainda não testada.

### Teste
Skill útil, mas ainda em validação.

### Ativa
Pode ser usada normalmente.

### Estável
Validada em uso recorrente e integrada ao sistema.

### Restrita
Usar apenas em contextos específicos.

### Obsoleta
Foi substituída ou perdeu validade.

### Arquivada
Preservada apenas para histórico.

## Registro

```markdown
# STATUS DE SKILL

## Nome
...

## Status
...

## Motivo
...

## Quando usar
...

## Quando não usar
...

## Substitui
...

## Substituída por
...

## Revisão futura
...
```

## Regra

Nem toda skill criada deve permanecer ativa para sempre.
