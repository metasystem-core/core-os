# Checklist de revisão documental — IRATA N3

## Documentos mínimos

- [ ] Procedimento de acesso por cordas.
- [ ] Plano de resgate específico.
- [ ] Plano de ancoragem.
- [ ] APR.
- [ ] PT.
- [ ] AF/autorização, se aplicável.
- [ ] Procedimento do cliente.
- [ ] Matriz de equipe.
- [ ] Matriz de equipamentos.
- [ ] Certificados/inspeções.
- [ ] Comunicação/emergência.
- [ ] Isolamento/queda de objetos.

## Critérios de bloqueio

- [ ] Resgate genérico ou inexistente.
- [ ] Ancoragem sem validação.
- [ ] Equipamento sem rastreabilidade.
- [ ] Equipe incompatível.
- [ ] APR/PT incompatível com atividade.
- [ ] Procedimento do cliente ignorado.
- [ ] Mudança de cenário sem revisão documental.
