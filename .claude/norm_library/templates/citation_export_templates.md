# Templates de Citação

## Curta

```text
{norm_code}, {edition}, item {item}, p. {page_printed}.
```

## Relatório técnico PT-BR

```text
Referência normativa: {norm_code} — {norm_title}, {edition}, item {item}, página {page_printed}.
```

## Fiscalização / cliente

```text
A base normativa utilizada é {norm_code}, {edition}, item {item}, página {page_printed} (PDF p. {page_pdf}). O item se aplica porque {operational_summary}. A evidência necessária é: {evidence_required}.
```

## Com limitação de vigência

```text
Referência: {norm_code}, {edition}, item {item}, página {page_printed}. Observação: a vigência desta edição está marcada como {validity_status}; confirmar edição contratual ou vigente antes de uso conclusivo.
```

## Com referência fraca

```text
Referência preliminar: {norm_code}, seção/página aproximada {page_printed}. Esta referência ainda não possui precisão suficiente para fundamento técnico forte; requer revisão no PDF original.
```
