# Sub-skill: Seleção de Técnica END

Consultada quando: escolher qual técnica END aplicar.

## Por localização do dano

| Localização | Técnicas indicadas |
|---|---|
| Superfície acessível | VT, PT, MT, ET |
| Subsuperfície (< 6mm) | MT, ET, UT |
| Interior / volumétrico | UT, RT |
| Parede inteira (espessura) | UT medição |
| Solda | RT, UT, MT, PT |

## Por material

| Material | Restrição |
|---|---|
| Aço carbono | Todas aplicáveis |
| Aço inoxidável | MT não funciona |
| Alumínio / titânio | MT não funciona — PT ou ET |
| Polímero | UT, termografia, VT |

## Por objetivo

| Objetivo | Técnica |
|---|---|
| Trinca superficial — ferromagnético | MT |
| Trinca superficial — não ferromagnético | PT |
| Medir espessura | UT |
| Inspecionar solda | RT ou UT |
| Detectar ponto quente | Termografia |
| Verificar delaminação | UT, termografia |

## Saída esperada

Dado mecanismo + material + acesso: recomendar técnica(s) em ordem
de adequação + justificativa + limitação para o caso específico.
