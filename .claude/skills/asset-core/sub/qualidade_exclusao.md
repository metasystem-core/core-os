# Qualidade da Exclusão

## Princípio

Avaliar a qualidade de uma decisão inclui avaliar a qualidade
das exclusões feitas. O que foi descartado e por quê é parte
do registro obrigatório.

## No ASSET-CORE

Uma tese boa não é só a que escolhe ativos corretos.
É a que exclui corretamente os ativos errados.

Registro obrigatório a cada decisão:
- O que foi considerado e excluído?
- Por que foi excluído?
- Qual seria o custo se a exclusão estiver errada?

LOTOFA-CORE — fórmula: Acertos = 5 + |E_c ∩ E_t|
O jogo bom não é só o que escolhe bem — é o que exclui certo.

## Onde pode falhar

Usar como justificativa para inação permanente.
Exclusão bem fundamentada é parte da decisão — não substituto dela.
