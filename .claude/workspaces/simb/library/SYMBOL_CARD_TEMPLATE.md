## Prohibited Uses

- Dizer "este símbolo significa [X]" sem qualificador
- Usar para diagnosticar estado do operador
- Prescrever ação com base no símbolo
- Afirmar com certeza

- Misticismo forçado ("o universo está dizendo que...")
