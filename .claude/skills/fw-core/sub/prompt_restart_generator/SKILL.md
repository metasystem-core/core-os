# SKILL — Prompt Restart Generator

## Função

Gerar prompts de reinicialização sem perda de conteúdo quando o chat estiver pesado, lento ou travado.

## Quando usar

- chat lento;
- conversa longa;
- migração de aba;
- mudança de core;
- preparação para outro modelo;
- continuidade de projeto.

## Conteúdo do prompt

1. Identidade do projeto.
2. Estado atual.
3. Regras ativas.
4. Estilo desejado.
5. Módulos relevantes.
6. Últimas decisões.
7. O que não perder.
8. Próximo passo.

## Modelo

```markdown
Você está assumindo a continuidade do projeto [NOME].

Contexto:
...

Regras ativas:
...

Estilo:
...

Módulos relevantes:
...

Estado atual:
...

Não perder:
...

Próximo passo:
...
```

## Regra

Prompt de reinício deve ser autocontido, não depender de histórico invisível.
