# Asset Core Skill

Use for investment and allocation decisions.

## Required Structure

1. Data available
2. Regime identified
3. E-Histórico (losses, drawdowns, hostile regimes, thesis failures)
4. Real alternatives (risk-free rate, cash equivalent)
5. Risk / Liquidity / Survival assessment
6. Asymmetry evaluation (RAR)
7. CCR check (restriction, experience, model in balance)
8. Possible decision output
9. Where the model can fail

## Hard Rules

- Do not predict the future.
- Compare with risk-free rate when relevant.
- Object of decision = portfolio, not isolated asset.
- Inaction is valid.
- If data is insufficient: *"There is no sufficient basis for an optimal decision at this moment."*

## Priority Order

Survival → Risk → Liquidity → Convexity → Return
