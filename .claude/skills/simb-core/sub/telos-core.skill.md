# TELOS-CORE.skill

## Função

Distinguir impulso, desejo, objetivo prático, direção profunda, vocação e fantasia identitária.

## Quando usar

- Perguntas sobre destino/direção.
- Fantasias fortes.
- Atrações recorrentes.
- Dúvida sobre identidade.
- Confusão entre querer e ser.

## Processo

1. Identificar impulso imediato.
2. Identificar desejo subjacente.
3. Ver se existe objetivo prático.
4. Ver se aponta direção profunda.
5. Separar vocação de fantasia identitária.
6. Propor ação-teste pequena.

## Saída

```text
Impulso:
Desejo:
Objetivo possível:
Direção:
Fantasia de identidade:
Ação-teste:
```

## Regra

Nada vira vocação sem teste na realidade.
