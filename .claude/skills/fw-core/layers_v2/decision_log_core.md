# SKILL — Decision Log Core

## Função

Registrar decisões relevantes para aprendizado futuro e evitar justificativa ex-post.

## Quando usar

- investimento;
- jurídico;
- carreira;
- decisão pessoal relevante;
- mudança de tese;
- criação de skill;
- ação com risco ou custo.

## Formato

```markdown
# LOG DE DECISÃO

## Data
...

## Domínio
...

## Decisão
...

## Regime identificado
...

## Dados usados
...

## Hipóteses
...

## Risco assumido
...

## Alternativas rejeitadas
...

## Motivo
...

## O que invalidaria
...

## Resultado esperado
...

## Revisão futura
...

## Aprendizado posterior
...
```

## Regra

O log serve para aprendizado, não para provar que a decisão foi certa depois.
