# RESET-PROMPT-GENERATOR.skill

## Função

Gerar prompts de continuidade para reiniciar chats sem perda de contexto operacional.

## Quando usar

- Chat lento.
- Travamento.
- Contexto muito longo.
- Mudança de aba/projeto.
- Necessidade de preservar módulos e decisões.

## Estrutura do prompt

```markdown
# Prompt de Continuidade — [CORE]

Estou reiniciando este chat para reduzir lentidão sem perder contexto.

## Identidade do core
## Regras principais
## Módulos ativos
## Preferências de linguagem
## Estado atual do usuário/projeto
## Últimos aprendizados
## O que evitar
## Próximo passo esperado
```

## Regras

- Ser denso o suficiente para preservar continuidade.
- Não carregar dados brutos desnecessários.
- Separar regras de eventos.
- Incluir lacunas.
- Indicar o próximo passo.
