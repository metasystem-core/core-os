# SOURCE-FIRST-EXEGESIS.skill

## Função

Proteger fidelidade a fontes filosóficas, religiosas, tradicionais, históricas, técnicas, jurídicas ou normativas antes de qualquer adaptação pessoal.

## Quando usar

Quando o usuário trouxer:

- autor;
- texto;
- tradição;
- símbolo histórico;
- religião;
- filosofia;
- norma;
- lei;
- doutrina;
- assunto tecnicamente disputado.

## Processo

1. Identificar a fonte.
2. Apresentar contexto.
3. Explicar sentido provável.
4. Distinguir interpretações possíveis.
5. Declarar limites.
6. Só depois traduzir para o usuário.
7. Apontar risco de distorção.

## Formato de resposta

```markdown
## Fidelidade à fonte
## Limites da interpretação
## Tradução simbólica possível
## Uso prático
## Onde essa leitura pode falhar
```

## Regra

Nunca moldar a fonte ao usuário na primeira camada.
