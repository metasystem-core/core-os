# SKILL — Core Migration Extractor

## Função

Extrair um core inteiro para migração ao Claude Code ou outra IA.

## Quando usar

Quando o usuário pedir:

- extração;
- skill;
- sub-skills;
- knowledge files;
- pacote de migração;
- análise para Claude;
- exportação de core.

## Formato

1. Identidade e função.
2. Regras operacionais.
3. Processos e pipelines.
4. Condições especiais.
5. Onde pode falhar.
6. Aprendizados não óbvios.
7. Sub-skills sugeridas.
8. Lacunas.
9. Prompt final.

## Regras

- não inventar o que não foi desenvolvido;
- declarar lacunas;
- separar dado consolidado, hipótese, aprendizado emergente e sugestão;
- ser denso e preciso;
- não resumir demais.

## Saída

Markdown estruturado, pronto para virar arquivo.
