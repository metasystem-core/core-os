# Checklist de captura de artefatos

## Para cada arquivo gerado

Registrar:

- nome;
- caminho;
- tipo;
- função;
- projeto/core relacionado;
- se é ZIP;
- se é all-in-one;
- se é técnico;
- se é documentação;
- se é prompt para Claude;
- se contém dado privado;
- próximo uso;
- status.

## Status possíveis

- generated;
- sent_to_claude;
- reviewed;
- integrated;
- superseded;
- archived;
- discard_candidate.

## Regra

Arquivo sem função registrada vira entulho.
