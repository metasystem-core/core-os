## Definition / Definição

Quando um número aparece num sonho — uma data, uma quantidade, uma idade, um endereço, um número repetido — ele é primariamente **sobredeterminado por associações pessoais**:

- Data de nascimento / morte de alguém significativo
- Endereço
- Número de meses ou anos de uma relação, trabalho, perda
- Quantidade de pessoas ou objetos com significado na história do operador
- Número de uma conta, telefone, resultado

**Regra operacional:** Exaurir a aritmética pessoal antes de qualquer amplificação de simbolismo coletivo. Só após confirmar que nenhuma associação pessoal resta é que se pode, com cautela, explorar o campo coletivo do número.

**Mesmo após associações pessoais esgotadas:** o simbolismo coletivo (7 como número de totalidade, 4 como quaterno, etc.) oferece um campo de possibilidades, não uma chave. O número coletivo informa a imagem, não a encerra.

**Risco específico (RISK-H-003):** Numerologia como sistema fechado. Se o operador chega com um sistema numerológico pré-formado ("é meu número de vibração, sempre aparece"), o simbolismo já está fixado antes das associações pessoais — PATCH-007 (anti-confirmação) aplica.

---

## Operational use / Uso operacional

- Quando número aparece no sonho: "O que este número evoca para você? É uma data, uma quantidade, um número que aparece na sua vida?"
- Percorrer sistematicamente antes de qualquer amplificação: datas, endereços, idades, durações, quantidades com significado pessoal.
- Quando associações pessoais forem abundantes: não há necessidade de amplificação coletiva — as associações pessoais são suficientes.
- Quando associações pessoais forem escassas e o número persistir em série: o campo coletivo pode ser explorado como hipótese, com declaração explícita de provisoriedade.
- EVAL-17 (número 7 repetindo + numerologia): cenário de teste desta regra.

---

## Do not use for / Não usar para

- Não usar para confirmar sistema numerológico do operador como chave interpretativa.
- Não usar para afirmar que o número tem "significado absoluto" baseado em tradição (cabalística, pitagórica, astrológica, etc.).
- Não iniciar interpretação de número pelo coletivo — isso é inversão do procedimento correto.
- Não tratar recorrência de número como confirmação de significado simbólico específico — recorrência é convite a mais associações, não prova.

## Historical note / Nota histórica

PH-004 está ancorado em Part I de *Dreams* (§§129–145), que está dentro do período histórico-freudiano de Jung (ver CON-043). O princípio de prioridade das associações pessoais, contudo, é metodologicamente estável no Jung maduro e não contradiz Parts II/III/IV. O card é promovido com status `medium` (não `high`) por reconhecer que a fonte primária é de Part I.

O procedimento pessoal-antes-de-coletivo é reforçado em Part II/III no contexto de amplificação geral (PP-023/024 — ver CON-038) e pode ser tratado como extensão metodológica do princípio mais amplo.
